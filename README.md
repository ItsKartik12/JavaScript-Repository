# JavaScript-Repository

A collection of JavaScript code snippets, practice problems, and implementations covering fundamental to advanced concepts. This repository serves as a personal reference for improving logic, understanding JS internals, and mastering modern syntax.

## 🚀 Overview

This repository contains organized solutions and explorations of JavaScript features. Whether you are reviewing core concepts or tackling algorithmic challenges, these files are structured to demonstrate clean, efficient, and modern code practices.

## 📂 Structure

The repository is organized by topic to make navigation straightforward:

*   **`/basics`**: Core language features (variables, loops, functions, scoping).
*   **`/data-structures`**: Implementations of common data structures (Arrays, Objects, Maps, Sets).
*   **`/algorithms`**: Problem-solving exercises with a focus on time and space complexity.
*   **`/es6+`**: Modern features including Promises, Async/Await, Destructuring, and Modules.
*   **`/utils`**: Helper functions and commonly used snippets for everyday tasks.

## 🛠 Tech Stack

*   **Language**: JavaScript (ES6+)
*   **Runtime**: Node.js (for testing and execution)

## 💡 Key Focus Areas

*   **Code Optimization**: Prioritizing time and space complexity in algorithms.
*   **Modern Practices**: Utilizing ES6+ features and clean code principles.
*   **Robustness**: Handling edge cases and focus on defensive programming.

## 📋 How to Use

1.  **Clone the repository:**
```bash
    git clone [https://github.com/ItsKartik12/JavaScript-Repository.git](https://github.com/ItsKartik12/JavaScript-Repository.git)
    ```
2.  **Run individual files:**
    Navigate to the specific folder and run the file using Node.js:
```bash
    node filename.js
    ```
